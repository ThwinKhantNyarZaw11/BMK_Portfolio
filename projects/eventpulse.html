EventPulse - Smart Event Manager

Overview

EventPulse is a web-based event management app designed for event organizers to manage participants, ticketing, and attendance through QR-based check-ins and analytics dashboards.

Key Features

  • QR code ticket generation and scanning system
  • Role-based admin panel with event performance metrics
  • Automated check-in and attendance tracking
  • Real-time stats: participant counts, drop-offs, demographics
  • Email confirmations and PDF ticket delivery

Technology Stack

  • Frontend: React, Tailwind CSS
  • Backend: Node.js, Express
  • Database: PostgreSQL
  • Libraries: qrcode, chart.js

Challenges & Solutions

  • Challenge: Preventing duplicate QR check-ins
  • Solution: Used token-based scanning with one-time validation logic
  • Challenge: Managing admin permissions securely
  • Solution: Implemented JWT roles and protected API routes

Impact

  • Enabled smoother management of medium-scale tech events
  • Reduced manual attendance errors and admin effort
  • Inspired future work in digital badge and certificate automation
