SafeVault - Encrypted File Storage App

Overview

SafeVault is a secure web application that enables users to upload and store files with end-to-end encryption. Featuring two-factor authentication, password-protected access, and dynamic download URLs, the app ensures user privacy and data integrity. This project demonstrates expertise in authentication, file handling, secure backend logic, and cloud-based storage strategies.

Key Features

  • End-to-end encrypted file uploads and AES-based file encryption
  • Two-factor authentication (2FA) using OTP via email
  • Unique, expirable download links with access control
  • Role-based dashboard with upload history and metadata
  • Self-destruct feature for files after one-time download

Technology Stack

  • Frontend: React, Tailwind CSS
  • Backend: Node.js, Express, JWT
  • Database: MongoDB (with GridFS for file storage)
  • Security: AES encryption, bcrypt, CORS, environment variables

Challenges & Solutions

  • Challenge: Secure file encryption with performance
  • Solution: Used streaming AES encryption with file chunks to reduce memory load
  • Challenge: 2FA integration without external services
  • Solution: Built a custom email-based OTP generator and validator with timeouts

Impact

  • Helped users confidently share sensitive files securely online
  • Applied full-stack principles with real-world encryption standards
  • Inspired future enhancements including biometric login and cloud integration
