RoadMapper - AI Career Planning Tool

Overview

RoadMapper is a smart web app that generates personalized learning paths for users based on their goals and current skillset. Using AI-powered suggestions and milestone tracking, users can build structured roadmaps in fields like software development, UI/UX, and data science.

Key Features

  • Interactive goal selection and skill assessment
  • Dynamic roadmap generation using OpenAI GPT API
  • Progress tracker and milestone reminders
  • User accounts with persistent roadmap storage
  • Export to PDF and share via link options

Technology Stack

  • Frontend: React, Tailwind CSS, Framer Motion
  • Backend: Node.js, Express
  • AI Service: OpenAI API (GPT 3.5)
  • Database: Firebase Realtime DB

Challenges & Solutions

  • Challenge: Keeping roadmap suggestions relevant across various fields
  • Solution: Tuned system prompts and filtered responses for category-specific output
  • Challenge: Saving user progress securely
  • Solution: Used Firebase auth with Firestore rules and real-time sync

Impact

  • Helped users build confidence and direction in self-learning journeys
  • Served as a prototype for future edtech personalization tools
  • Enabled feature reuse in future goal-tracking projects
