MediNote - Secure Medical Notes App

Overview

MediNote is a healthcare-focused note-taking application built for doctors and healthcare professionals. It provides organized patient tagging, encrypted notes, and custom exports to streamline clinical documentation in secure environments.

Key Features

  • Patient-linked note system with automatic sorting
  • Secure storage with end-to-end note encryption
  • Export to PDF for physical records or sharing
  • Search and filter by patient name, condition, or tags
  • Dark mode for clinical environments

Technology Stack

  • Frontend: React, Tailwind CSS
  • Backend: Firebase Auth + Firestore
  • Export: PDFMake, html2canvas
  • Security: CryptoJS for encryption, role-based access

Challenges & Solutions

  • Challenge: Securing sensitive medical notes on client side
  • Solution: Used AES encryption with user-specific secret keys
  • Challenge: Designing UI for fast notetaking with minimal clicks
  • Solution: Built keyboard-accessible layout with quick-tagging UX

Impact

  • Improved speed and structure of daily patient note workflows
  • Demonstrated secure frontend data handling practices
  • Used in prototypes for mobile-first digital health tools
