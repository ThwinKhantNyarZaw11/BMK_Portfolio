CodeBoard - Collaborative Code Editor

Overview

CodeBoard is a real-time, collaborative code editor inspired by pair programming. It allows multiple users to edit code simultaneously in a shared online workspace with syntax highlighting and integrated terminal output.

Key Features

  • Live code collaboration with WebSocket sync
  • Supports multiple languages with syntax highlighting
  • Integrated output preview and terminal
  • Custom room sharing via unique link
  • Dark/light theme toggle and user cursor indicators

Technology Stack

  • Frontend: React, Tailwind CSS
  • Editor: Monaco Editor (VSCode base)
  • Realtime: Socket.IO
  • Backend: Node.js + Express

Challenges & Solutions

  • Challenge: Handling edit conflicts and race conditions
  • Solution: Implemented debounced sync and authoritative server state
  • Challenge: Integrating editor with multi-language execution
  • Solution: Built Docker-based code runner microservice

Impact

  • Enabled remote coding interviews and live tutoring
  • Used by students for project collaborations and review sessions
  • Extended into hackathon-ready demo environments
